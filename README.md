# Ping Pong Game (Python Turtle)

A simple two-player Ping Pong game built using Python and the `turtle` module.

## Controls
- Left paddle: `W` / `S`
- Right paddle: `↑` / `↓`

## Features
- Two-player local gameplay
- Ball and paddle collision
- Score tracking
- Smooth game loop

## Requirements
- Python 3

## Run
```bash
python ping_pong.py
